  • executeRequest(httpMethod:_:targetUrl:requestObject:headers:success:failure:retryAttempts:)

    Performs a httpMethod kind request to the given path. If the requests succeeds, the completion block will be called after converting the result using the given transformer. If the request fails, the ‘errorHandler’ block will be called instead.
